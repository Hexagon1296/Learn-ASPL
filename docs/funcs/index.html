Libraries

Libraries contain extra functions that can be imported using an import statement: import ...

If you want to, you can import only specific functions using a from import statement: from ... using (...) (The functions to import have to be in brackets, due to techical limitations)

If you think a library's name is too long or just not good, you can use a different name with named: import ... named ... It is not possible to combine it with a from import statement, so you can't import a function using different name
